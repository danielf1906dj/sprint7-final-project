# sprint7-final-project
Análisis de una empresa de telecomunicaciones
# Objetivo del proyecto

El objetivo de este proyecto es analizar el comportamiento de uso de los clientes de ConnectaTel (empresa de telecomunicaciones en Latinoamérica) para identificar patrones de consumo, detectar valores atípicos y segmentar usuarios según su comportamiento.

A partir de este análisis, se busca generar insights accionables que permitan mejorar la oferta de planes, optimizar la experiencia del cliente y apoyar la toma de decisiones estratégicas.

# Datasets utilizados
- plans.csv
- users_latam.csv
- usage.csv
  
# Etapas del análisis
- Carga y exploración de datos
- Revisión Calidad de datos
- Limpieza de datos
- Transformación y agregación
- Detección de outliers
- Segmentación de clientes
- Insights de negocio
